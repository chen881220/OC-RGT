According to your system envrionment to change the Config.xml file.

Windows:
  Open command prompt as administrator, and type python main.py [--autoreport] [--installer]
  [--autoreport]:
    Test framework will automatically send the result report when finish.
  [--installer]:
    Test framework will atuomatically download the newest installer and then install the newest installer.

Linux:
  chmod a+x Run.sh
  sudo ./Run.sh
